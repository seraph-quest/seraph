# Retired Village Editor

The standalone village map editor is retired.

It is no longer part of the supported Seraph product surface or roadmap.

Why this file is now minimal:

- Seraph is cockpit-only
- the village/editor line is historical code pending removal
- there is no value in maintaining detailed operating docs for a surface we do not plan to ship

If you need the older implementation details for historical recovery work, use git history for this folder rather than treating it as a supported subsystem.
